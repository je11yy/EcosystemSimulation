### Шаблон коммита

`<type> <workPart>: <description>`

- ```<type>``` - тип коммита
- ```<workPart>``` - часть проекта, где были внесены изменения
- ```<description>``` - описание изменений

---

### Типы коммитов:
- `feat` - новый код, добавление чего-то нового в проект
- `fix` - исправление ошибок
- `docs` - написание комментариев, документации, изменение readme, gitignore
- `format` - форматирование кода

---

### Части проекта
- `frontend` - папка frontend
- `server` - папка server
- `core` - папка simulationCore
- `other` - другое
